# AnyTime

A timezone calculator for iOS and macOS, built with SwiftUI.

- Search by city, region, abbreviation, and UTC offset
- Smart paste that can extract date and timezone hints from the clipboard
- Drag reordering, swipe actions, quick `+/-1h` and `+/-1d` controls
- Dark mode support

Targets iOS 17.0+ and macOS 14.0+.

## Install

### Homebrew

```bash
brew tap hewigovens/tap
brew install --cask anytime
```
